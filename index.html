Project Overview

Background

Personalised nutrition is an emerging and rapidly expanding segment of the health and wellbeing industry. An increasing number of apps and devices are being produced to address potential market demand. Inner Intel (working title) is a personalized nutrition app that distinguishes itself from others on the market1 in two ways: it is not prescriptive, and; it uses a machine learning AI to analyse supplied personal data.

Goals for 2022, S1

General client-side Design

A simple clear style interface(UI) which closely related.The APP can meet the needs of users to easily input daily diet. Try to establish the classification model or regression model of user data.

Nutritionist side Design

We will design a user-friendly website interface. Nutritionists can easily find and analyze data which is already well-organized. At the same time, we will provide a communication platform, so that they can better communicate with customers.

Management side Design

We provide administrators with a data platform that allows them to monitor current users and dietitians in real time and process daily feedback.The purpose of this is to facilitate timely adjustment and provide a better APP environment.

Introduction to Personalised Nutrition Project

What functions dese "Inner Intel" have?

Record Diet

It provides users a simple and friendly way to input their daily eatings and body responses, uses these data to do some analyzes.

Give Advice

It can generate timely advice to the users, in order to help them stay healthy. This advice will be based on the user's personal diet, physiological responses and health habits.

Generate Report

It can generate a report to show some basic data to nutritionists. Of course, it can extract key information from vast amounts of health data to generate some general guidance.

Connect Nutritions

It can provide methods to connect users with nutrition experts, if the users want to improve their daily nutrition formula.

Why use "Inner Intel"?

Personalized

It can help track users' daily diet and provide recommendations tailored for every user.

Timely remind

It's a reminder to change users' diet or behavior when they are at risk.

Exclusive

It will generate a healthy report based on the certain user individually.

Professional

Users can get advice from several nutrition experts online.

Easier input

Users can easily enter personal information by taking photos, selecting lists, selecting ICONS, and typing text.

Collective

Users information can be collected and well-organized for dietitians.

Interactive

Users can communicate with nutrition experts anytime, anywhere as well as other users.
