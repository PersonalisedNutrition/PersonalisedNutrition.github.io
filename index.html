Project Goals for 2022 , 1

General client-side Design

A simple clear style interface(UI) which closely related.The APP can meet the needs of users to easily input daily diet. Try to establish the classification model or regression model of user data.

Nutritionist side Design

We will design a user-friendly website interface. Nutritionists can easily find and analyze data which is already well-organized. At the same time, we will provide a communication platform, so that they can better communicate with customers.

Management side Design

We provide administrators with a data platform that allows them to monitor current users and dietitians in real time and process daily feedback.The purpose of this is to facilitate timely adjustment and provide a better APP environment.

Introduction to Personalised Nutrition Project

Inner Intel is a health app that focuses on each individual

1. It provides users a simple and friendly way to input their daily eatings and body responses, uses these data to do some analyzes,

2. It can generate timely advice to the users, in order to help them stay healthy. This advice will be based on the user's personal diet, physiological responses and health habits.

3. It can generate a report to show some basic data to nutritionists. Of course, it can extract key information from vast amounts of health data to generate some general guidance.

4. It can provide methods to connect users with nutrition experts, if the users want to improve their daily nutrition formula.

Why use it?

Personalized

It can help track users' daily diet and provide recommendations tailored for every user.

Timely remind

It's a reminder to change users' diet or behavior when they are at risk.

Exclusive and Professional

It will generate a professional healthy report based on the certain user individually.

Easier input and Collective

Users can easily enter information by many ways.Information can be collected and well-organized.
