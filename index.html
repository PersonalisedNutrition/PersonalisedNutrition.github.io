Project Overview

Background

Personalized nutrition is an emerging and rapidly expanding segment of the health and wellbeing industry. An increasing number of apps and devices are being produced to address potential market demand. Inner Intel (working title) is a personalized nutrition app that distinguishes itself from others on the market1 in two ways: it is not prescriptive, and; it uses a machine-learning AI to analyze supplied personal data.

However, there is a great difference between people's physical constitution, and there may be great differences in allergens, eating habits and living habits. If general conclusions are used, they may not be well adapted to current users. After evaluating some other products, we found that the apps currently on the market for the most part are not good for users with personalized settings. They can't give users advice according to their current health status. They just give some general advice based on the results of the statistics and tend to avoid giving too in-depth advice in order to prevent other problems caused by individual differences among users.

Inner Intel (working title) is a personalized nutrition app that distinguishes itself from others on the market in three ways: The app will make personalized suggestions based on each person's different situation; It uses a machine-learning AI to analyze supplied personal data; Users can have one-to-one communication with dietitians, and our app will make personalized adaptations according to the communication results.

Goals for 2022, S2

Web design for the nutritionist

Considering that nutritionists prefer to work in front of computers. This semester we will design a web for nutritionists, which consists of a friendly interactive web front-end and a web back-end for data processing. With the web, nutritionists can register and log in, communicate with users, push suggestions to clients of specified users, and set some items for them.

Designing a database

For the convenience of users to store information, but also for the convenience of nutritionists to check the status of users, timely advice. We will build a database this semester to store records and reports separately for each client and establish an appropriate relationship schema. And we will use questionnaires to collect initial user data.

Introduction to Personalised Nutrition Project

What functions does "Inner Intel" have?

For users:

Record Diet

It provides users a simple and friendly way to input their daily eating and body responses, uses these data to do some analyzes.

Know yourself

The app's intelligent analytics can give you a better sense of how your body responds to your daily habits.

Get advice

Users can get advice from nutritionists to change the state of health management from disorganized to organized.

For nutritionists:

View the data

Nutritionists can easily view the data of a specific user through the Web, analyze it and make recommendations to the user.

One-on-one communication

The nutritionist can communicate one-on-one with a specific user and set up the client for that user based on the results.

Report review and release

Nutritionists can view user reports generated by the system or push their own analysis reports to users.

Why use "Inner Intel"?

For users:

Personalized

It can help track users' daily diet and provide recommendations tailored for every user.

Timely reminder

It's a reminder to change users' diets or behavior when they are at risk.

Exclusive

It will generate a healthy report based on the certain user individually.

Professional

Users can get advice from several nutrition experts online.

Easier input

Users can easily enter personal information by taking photos, selecting lists, selecting ICONS, and typing text.

Collective

Users' information can be collected and well-organized for dietitians.

Interactive

Users can communicate with nutrition experts anytime, anywhere as well as other users.

For nutritionists:

Easy Communication

Nutritionists can easily communicate with their clients through the Web.

Well Formed Data

User data is well structured and stored for analysis by nutritionists.

Custom Client End

The nutritionist can directly set up the user's client based on the results of communication with the user.

Report Review And Release

Nutritionists can view the reports the system generates for each user and can push their own analysis reports to the user.
